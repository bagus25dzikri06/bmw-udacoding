An element with a static position is always positioned according the normal flow of the web page.

This element has a static position

An element with a relative position is positioned relative to its normal position.

An element with an absolute position is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like the fixed one).

This element has a relative position
This element has an absolute position

An element with a fixed position is positioned relative to the viewport, where it always stays in the same place even when the web page is scrolled.
